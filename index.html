Neocast

A Mindful of Monsters

A Mind of Monsters is a 3D Roguelike story driven sword game.
In this project, I made this project with an artist friend and I was responsible for Gameplay / UI programming and abilities upgrade system where players can use collected experience to power up abilities.

  • Link

Gain Access (Hacking Game)

Gain Access is a Top-down 2D hacking game. The inspiration for this project came from one of my favorite television-series MR.ROBOT.
I made this project as part of an online hackathon. For this project, I coded an on-screen hacking device that allowed players to unlock quests (perform attacks) and other gameplay features such as character movement and other in-game electronics device.

  • Link

Boxing Bag Piano

Boxing Bag Piano is my Ludum Dare game.
This game is more-like a casual time based clicker game where players have to click on boxing when it comes in the same area of shape. For this project, I was responsible for bit of everything: Art, Sounds and Programming. I coded gameplay logic, UI and designed a few 2D sprites in Photoshop along with sound effects and background music.
